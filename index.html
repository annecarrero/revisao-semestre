Conteúdos Adquiridos em Sala de Aula

Conteúdo Descrição Pontos Positivos Pontos Negativos
HTML Estruturação básica de páginas web. Fácil de aprender, essencial para web. Limitado em termos de funcionalidades dinâmicas.
CSS Estilização de páginas web. Permite designs visualmente agradáveis. Pode ser complexo para designs avançados.
JavaScript Linguagem de programação para interatividade. Flexível e poderoso. Problemas de compatibilidade entre navegadores.
Node.js JavaScript no lado do servidor. Alta performance, bom para I/O intensivo. Menos adequado para operações CPU-intensivas.
Máquina Virtual Ambiente de desenvolvimento isolado. Facilita testes e desenvolvimento seguro. Consome muitos recursos do sistema.
React Native Desenvolvimento de apps móveis com JavaScript. Desenvolvimento rápido e eficiente. Performance pode ser inferior a apps nativos.
